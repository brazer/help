Основное описание

CreateShapeGIS – это приложение для просмотра сейсмических данных по Земле и Беларуси. Эти данные в приложении отображаются в виде таблицы и слой точек на карте. Приложение было разработано на языке Visual Basic с использованием элемента управления ActiveX MapWinGIS.
