Фильтрация данных (Filter)

Для отбора сейсмических данных существует форма (окно) Filter. Чтобы вызвать это окно, необходимо в меню Commands выбрать команду Filter…. Так как структуры данных по Земле и по Беларуси отличаются, то в зависимости от региона загружается соответствующее окно.

Критерий отбора по земле может устанавливатся по следующим параметрам:

  • по дате (год, месяц, день);
  • по магнитуде (MPSP, MPLP, MS или по любому значению магнитуды);
  • по глубине (в км).

Критерий отбора по Беларуси может формироватся по следующим параметрам:

  • по дате (год, месяц, день);
  • по магнитуде.

При нажатии кнопки OK будут загружены данные по указанному критерию. При этом форма будет закрыта. А при нажатии кнопки Accept также будут загражены данные, но окно при этом не закрывается. При нажатии кнопки Cancel окно будет закрыто.

Как можно заметить, не все параметры можно использовать в критерии. Для их активации нужно установить галочку.

В области даты существуют два варианта её определения.


Первый вариант позволяет указывать дату не полностью (в отличие от второго варианта), например только по годам.

Пример. Найдём сейсмические данные по Земле за период 2010-2012 гг. по магнитудам от 5 до 7 по глубине менее 71 км.

Ход действий (см. рис. ниже):

  1. При необходимости перейти в режим работы с регионом Земля. Для переключения нужно использовать меню Commands->Switch->Earth.
  2. Загрузить окно Filter (Commands->Filter…).
  3. Для ввода годов будем использовать первый вариант ввода даты. В поле From введём значение "2010", а в поле To - "2012".
  4. Для активации области ввода магнитуды установим флажок в поле Magnitude. Оставим переключатель на значение "MPSP". В поле From введём значение "5", а в поле To - "7".
  5. Установкой флажка в поле Depth активируем область ввода глубины. В поле From введём значение "0", а в поле To - "71".
  6. Нажмём на кнопку OK.

В результате этих действий будут загружены сейсмические данные по указанному критерию.
